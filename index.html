Project: Serverless File Sharing & Real-Time Collaboration Platform

The Problem

Many file-sharing tools feel too heavy or slow for quick collaboration. I wanted to build a simpler platform where users can upload files, share them securely, and comment in real time.

What I Built

  • Secure file upload and download
  • User-based file sharing
  • Real-time comments
  • Authentication and access control
  • Monitoring with CloudWatch

Tech Stack

AWS S3, Lambda, DynamoDB, AppSync, Cognito, CloudWatch, Amplify, GitHub CI/CD, GraphQL

Product Decisions

  • Used serverless architecture to reduce cost and avoid managing servers.
  • Used AppSync because it supports real-time GraphQL subscriptions.
  • Used S3 presigned URLs to make uploads faster and more efficient.

Challenges

  • Fixed GraphQL subscription issues.
  • Improved access control for sharing files.
  • Used CloudWatch logs to debug errors.

Outcome

I delivered a working MVP from idea to deployment, showing ownership across product thinking, cloud architecture, testing, and iteration.

What I Would Improve Next

  • Add product analytics
  • Improve onboarding
  • Add AI-powered file summaries
  • Create a mobile-first experience
